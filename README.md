# swing-mini-asangalyeva
Решение задач на Swing 

Мини-проекты на Java

В данном репозитории находятся 5 мини проектов
В процессе выполнения данной работы были использованы основные компоненты Java Swing, такие как JFrame, JButton, JPanel, JLabel, JTextField, JList, Timer, Graphics2D и различные обработчики событий и так далее


Project1 -Calculator:
в данном проекте был  создан графический калькулятор, он позволяет выполнять базовые математические операции: сложение, вычитание, умножение и деление. Для интерфейса использовались кнопки, поле отображения результата и сеточная компоновка элементов. Также была реализована логика обработки пользовательского ввода и отображения результата вычислений.

Project02 - To-Do List(список задач)
Во втором мини-проекте-это для управления списком задач. Пользователь может добавлять новые задачи, удалять их двойным нажатием мыши и очищать список

Project3 - Guess Number, то есть в этом проекте Угадывают загаданнык числа
Так скажем мини-игра «Угадай число». Программа случайным образом загадывает число от 1 до 100, а пользователь должен угадать его. После каждой попытки приложение выводит подсказку: загаданное число больше или меньше введённого значения. Также реализован подсчёт количества попыток и возможность начать игру заново.

Project04 - Drawing App(Рисовалка)
В этом проекте была создана простая рисовалка, в котором пользователь может рисовать мышью на панели приложения, изменять цвет линии и очищать его холост. В проекте использовалась работа с графикой, перерисовкой компонентов и обработкой движения мыши.

Project05 - Stopwatch(Tаймер/Секундомер)
Данный мини проект представляет собой секундомер с графическим интерфейсом, в нем реализованы кнопки запуска, паузы и сброса времени. Таймер обновляет отображение времени каждую секунду в формате MM:SS. Проект был создан для практики работы с Timer и управлением состоянием приложения.


# Как запустить проектыЖ  
Необходимо открыть проект в IntelliJ IDEA, можно можно скачать в ZIP-формате с GitHub данное репо или открыть, либо склонировать репозиторий через Git и открыть у себя в IntelliJ IDEA
После открытия проекта необходимо выбрать нужный проект и запустить файл `Main.java`
